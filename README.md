This project involves the design and verification of a parameterized Universal Asynchronous Receiver/Transmitter (UART) implemented in synthesizable Verilog HDL. The design operates at a configurable baud rate and is structured into three core modules: a baud rate generator (baudrate), a transmitter (u_xmit), and a receiver (u_rec). The transmitter serializes parallel data using a Moore FSM with states IDLE, START, WAIT, TX, and STOP, while the receiver reconstructs bytes from the serial line through its own FSM. Standard UART framing is used — 1 start bit, 8 data bits, and 1 stop bit — with each bit period spanning 16 uart_clk ticks for accurate mid-bit sampling.
Verification was carried out using a self-checking testbench that ran the DUT and a golden reference model (uart_ref_model) in parallel, applying identical stimuli and comparing outputs automatically. The testbench covered 19 feature-ID test cases including normal byte transmission and reception, boundary values (0x00, 0xFF), framing error injection, missing start-bit scenarios, mid-frame data changes, and mutual-exclusion checks on status signals. A key fix was synchronizing comparisons to wait for both DUT and reference to complete each frame, eliminating false mismatches on back-to-back transfers.
Simulation in Questa SIM yielded a total code coverage of 93.94%, with 95.45% on statements,95.45% on branches,90.00% on FEC conditions,100.00% on FEC Expressions,99.45% on toggles,83.33% on FSM as a whole,100.00% on FSM states and 66.66% on FSM Transitions.Out of 18 test cases, 18 passed and 0 failed (100.00% pass rate). Future work includes random test generation, SVA formal verification, parity support, multi-baud testing, and eventual SoC integration.
