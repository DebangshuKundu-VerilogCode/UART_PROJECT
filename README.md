This project involves the design and verification of a parameterized Universal Asynchronous Receiver/Transmitter (UART) implemented in synthesizable Verilog HDL. The design operates at a configurable baud rate (default 2400 baud at 100 MHz) and is structured into three core modules: a baud rate generator (baudrate), a transmitter (u_xmit), and a receiver (u_rec). The transmitter serializes parallel data using a Moore FSM with states IDLE, START, WAIT, TX, and STOP, while the receiver reconstructs bytes from the serial line through its own FSM. Standard UART framing is used — 1 start bit, 8 data bits, and 1 stop bit — with each bit period spanning 16 uart_clk ticks for accurate mid-bit sampling.
Verification was carried out using a self-checking testbench that ran the DUT and a golden reference model (uart_ref_model) in parallel, applying identical stimuli and comparing outputs automatically. The testbench covered 19 feature-ID test cases including normal byte transmission and reception, boundary values (0x00, 0xFF), framing error injection, missing start-bit scenarios, mid-frame data changes, and mutual-exclusion checks on status signals. A key fix was synchronizing comparisons to wait for both DUT and reference to complete each frame, eliminating false mismatches on back-to-back transfers.
Simulation in Questa SIM yielded a total code coverage of 90.02%, with 100% on statements, branches, FEC conditions, and FSM states. Toggle coverage came in at 64.83% and FSM transition coverage at 70.58%, reflecting some untriggered edge paths. Out of 18 test cases, 6 passed and 12 failed (33.33% pass rate), indicating areas for further debugging and refinement. Future work includes random test generation, SVA formal verification, parity support, multi-baud testing, and eventual SoC integration.
